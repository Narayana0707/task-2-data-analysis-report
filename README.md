# Task 3 – SQL Basics: Filtering, Sorting & Aggregations

## Dataset
Chinook Database

## Tools Used
- MySQL / PostgreSQL / SQLite
- SQL Workbench or pgAdmin

## Objective
To practice basic SQL concepts such as filtering, sorting,
aggregations, and grouping on a real dataset.

## Queries Performed
1. Viewed sample data using SELECT
2. Verified record count using COUNT
3. Filtered data using WHERE
4. Sorted results using ORDER BY
5. Calculated total and average sales using SUM and AVG
6. Grouped data using GROUP BY
7. Filtered grouped data using HAVING
8. Used BETWEEN for date range filtering
9. Used LIKE for pattern matching
10. Found top 5 customers using JOIN and aggregation
